# 📚 StudyFlow - Smart Study Planner

A comprehensive, student-focused study planner web application that transforms chaotic study schedules into organized, actionable plans.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📝 **Subject Management** | Add, edit, and prioritize subjects with syllabus load and exam dates |
| ⚡ **Smart Schedule Generation** | Automatically distributes study hours based on priority and deadlines |
| 📊 **Interactive Dashboard** | Real-time KPIs including overdue tasks, completed hours, and study streak |
| ✅ **Daily Checklist** | Mark tasks complete with visual progress tracking |
| 🔄 **Smart Reschedule** | One-click rescheduling to optimal time slots |
| 🎯 **Focus Timer** | Built-in Pomodoro timer with customizable sessions |
| 📅 **Deadline Management** | Track exams and identify high-risk subjects |
